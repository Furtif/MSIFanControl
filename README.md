# 机械革命S1（MSI Laptop） for macOS Control Fan

## 扣扣交流群：[45822629](https://jq.qq.com/?_wv=1027&k=5F5QsbT)

## Features

* SSDT-FANQ.dsl
   * Hotpatch / ACPI补丁,手动编译后添加到EFI

* MSIFanService
   * kext / 驱动服务,编译后添加到EFI

* MSIECControl
   * Program / 无GUI可执行二进制程序,编译后以命令形式执行

* MSIFanControl
   * App / GUI控制程序,目前还有Bug(设置后无效)
